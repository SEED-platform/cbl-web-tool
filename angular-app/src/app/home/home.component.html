Covered Building List Tool

Welcome to the CBL Editor, a tool for creating or managing Covered Building Lists (CBLs). With this editor, you can generate a new CBL from various formats, including CSV, Excel, JSON, or GeoJSON files. To create a comprehensive CBL, ensure that your data includes address, city, and state information.

The editor provides intuitive features to modify and enhance your CBL data. You can easily edit existing entries, delete records, or add new buildings. Additionally, a dynamic map view displays all the buildings from your CBL, allowing for efficient and visual management of your data

Requirements
  • A valid MapQuest API key
  • File(s) in a valid format(XLSX, CSV, JSON, GeoJSON)
  • File(s) MUST contain a Address, City, and State for each entity being represented
  • Incomplete or invalid data can distort results. Ensuring data accuracy and completeness is essential for reliable outcomes
